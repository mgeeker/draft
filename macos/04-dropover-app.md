### 背景
MacOS系统，在用户目录里拖动文件是移动文件，而大部的时候我们目标目录是没有打开，或是很多人并不知道怎么开两个finder
- Command + t 给finder开一个Tab
- Command + n 开一个新的finder

当你已经拖住文件的时候怎么开Tab或是新的Finder？

好无助~~~~

dropover来解决，给你的移动文件加一个“菜篮子”

这个神奇的“菜篮子”有好多好多你想不到的功能
